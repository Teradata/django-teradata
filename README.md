# Experimental readonly Teradata backend for Django

## Usage


Configure the Django `DATABASES` setting similar to this:

```python
DATABASES = {
    'default': {
        'ENGINE': 'django_teradata',
        'NAME': 'MY_DATABASE',
        'HOST': 'hostname[:port]',
        'USER': 'my_user',
        'PASSWORD': 'my_password'
    }
}
```

The backend enforces ANSI mode and TD2 logmech.
